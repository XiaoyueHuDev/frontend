# 轮播图组件
## API

属性说明如下：

| 属性 | 说明 | 类型 | 默认值 | 是否必填|
| --- | --- | --- | --- | --- | --- |
| imgSrc | 图片资源数组 | array | - | 是 |
| imgWidth | 图片宽度  | number  | 父元素宽度 |  否 |
| imgHeight | 图片高度  | number  | 父元素高度 |   否 |
| duration | 图片动画完成需要的时间 | number | 800 | 否  |
| autoTimer | 自动切换时间 | number | 3000 | 否 |
